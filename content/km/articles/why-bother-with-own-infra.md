бһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҹбһҪбһҡбһҒбҹ’бһүбһ»бҹҶбһҗбһ¶ бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҒбҹ’бһүбһ»бҹҶбһҹбһ»бһҒбһ…бһ·бһҸбҹ’бһҸбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ Deploy website TFDevs бһҸбһ¶бһҳбһҡбһҷбҹҲ Cloudflare, Cloudflare Tunnel, Nginx gateway VM, бһ“бһ·бһ„ K3s Kubernetes cluster бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ»бһ… "Deploy" бһҳбһҪбһҷбһғбҹ’бһӣбһёбһҖбһӣбһҫ Vercel бһұбҹ’бһҷбһ…бһ”бҹӢбһҡбһҝбһ„бһ‘бҹ…?
бһҖбһ¶бһҡбһ–бһ·бһҸбһ‘бҹ… Vercel бһӮбһәбһҮбһ¶ platform бһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҳбһҪбһҷ бһ”бһҫбһӮбҹ„бһӣбһҠбҹ…бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ„бҹӢбһұбҹ’бһҷ website бһ“бҹҒбҹҮ online бһ”бһ¶бһ“бһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸ бһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһҫбһңбһ¶бһ”бһ¶бһҸбҹӢбһ бһҫбһҷбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҒбҹ’бһүбһ»бҹҶ бһӮбҹ„бһӣбһҠбҹ…бһӮбһәбһ’бҹҶбһҮбһ¶бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ Hosting бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ” бһҖбҹ’бһ“бһ»бһ„бһ“бһ¶бһҳбһҮбһ¶ Software Engineer, Instructor, бһ“бһ·бһ„ Content Creator бһҒбҹ’бһүбһ»бҹҶбһ…бһ„бҹӢбһҷбһӣбҹӢбһұбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷ (beneath the surface) бһ“бҹғ Web applications бһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбҹ” Managed platforms бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҸбҹӮбһ„бһҸбҹӮбһӣбһ¶бһҖбҹӢбһ”бһ¶бҹҶбһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹғ Networking, Reverse proxies, Load balancing, Container orchestration, Monitoring, Scaling бһ“бһ·бһ„ Securityбҹ” бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Infrastructure бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһҮбһҪбһҷбһұбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (Hands-on) бһ“бһ·бһ„бһҷбһӣбҹӢбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ System бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„ Production бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

бһҒбҹ’бһүбһ»бҹҶбһҷбһӣбҹӢбһҗбһ¶ бһңбһ¶бһҳбһ¶бһ“бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһӣбһҫ Platform бһҠбһјбһ…бһҮбһ¶ Vercel, Netlify бһ¬ Managed services бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” Platform бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһ¶ Tool бһҠбҹҸбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһ“ Productivity бһҖбҹҸбһ–бһ·бһҸбһҳбҹӮбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶ "бһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹҶбһ—бһ¶бһ“бҹӢ" (Illusion) бһҗбһ¶ Web application бһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ Push бһ…бһјбһӣ Git бһҡбһҪбһ…бһңбһ¶бһҖбҹҸбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ Website бһҠбҹ„бһҷбһҳбһ“бҹ’бһҸбһўбһ¶бһӮбһҳбҹ” Magic бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһҖбһҫбһҸбһЎбһҫбһ„ бһҳбһ·бһ“бһҗбһ¶бһҮбһ¶бһ”бһүбҹ’бһ бһ¶ Networking, Performance bottlenecks, Scaling, SSL configuration, Reverse proxy errors бһ¬ Infrastructure outages бһ“бҹ„бҹҮбһ‘бҹҒ Developer бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бһ№бһ„бһ–бһ·бһ”бһ¶бһҖбһҷбһӣбҹӢбһҗбһ¶бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһҶбһ¶бһҖбҹ” бһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣ бһ–бһҪбһҖбһӮбҹҒбһ“бһ№бһ„бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫ Platform бһҺбһ¶бһҳбһҪбһҷ (Platform dependent) бһ“бһ·бһ„бһҮбһҪбһ”бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһ“бҹ…бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһң Deploy application бһҖбҹ’бһ“бһ»бһ„ Environment бһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“ Abstractions бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ—бһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһ”бҹӮбһ”бһ“бҹҒбҹҮбҹ”

бһҖбһ¶бһҡбһҠбһҫбһҡбһӣбһҫбһ•бҹ’бһӣбһјбһң Infrastructure бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһ“бҹҒбҹҮ бһҮбһҪбһҷбһұбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһҡбҹҖбһ“бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ–бһёбһҡбһ”бҹҖбһ” Build application бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһҡбһ”бҹҖбһ” Operate бһңбһ¶бһ‘бҹҖбһҸбһ•бһ„бҹ” бһҡбһ¶бһӣбҹӢ Component бһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„ Stack бһ“бҹҒбҹҮ бһ”бһ„бҹ’бһҡбҹҖбһ“бһҒбҹ’бһүбһ»бҹҶбһ“бһјбһңбһҳбҹҒбһҡбҹҖбһ“бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҸбһ¶бҹҶбһ„бһ–бһё Kubernetes, Ingress controllers бһҡбһ бһјбһҸбһҠбһӣбҹӢ Observability, Networking бһ“бһ·бһ„бһҖбһ¶бһҡ Troubleshooting distributed systemsбҹ” Homelab бһ“бҹҒбҹҮбһҖбҹҸбһҮбһ¶ "бһҖбһ“бҹ’бһӣбҹӮбһ„бһӣбҹҒбһ„" бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҒбҹ’бһүбһ»бҹҶбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ Technology бһҗбҹ’бһҳбһёбҹ—, Host project бһ•бҹ’бһҹбҹҒбһ„бҹ— бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё Concept бһ“бҹғ Real-world infrastructure бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбһ·бһҹбҹ’бһҹбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡ Maintenance бһ“бһ·бһ„бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ Managed platform бһҖбҹҸбһҠбҹ„бһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӮбһәбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҶбҹ’бһ„бһ¶бһҷбһҺбһ¶бһҹбҹӢбҹ” Website бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһ¶ Application бһҳбһҪбһҷбһҠбҹӮбһӣбһҡбһҸбҹӢбһӣбһҫ Infrastructure бһ“бҹҒбҹҮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҸбҹӮ Project бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶ бһӮбһәбһҖбһ¶бһҡбһҡбҹҖбһ“ бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбҹҶбһ“бһ¶бһү Engineering бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Modern systems бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбҹ”

សរសេរកូដដូចជាការសង់ផ្ទះ សង់ហើយត្រូវមើលថែទាំ Maintenance, Update , Fix Bug
កាលណាផ្ទះមានមនុស្សនៅ Project យើង មានអ្នកប្រើច្រើន ស្មើនឹងយើង មាន បំណុលបច្ចេកទេស (Technical Debt) ដែលត្រូវតាម Maintenance រហូត។ ដូច្នេះហើយទើប Software Engineer ត្រូវចេះសរសេរ Documents ពី Project ដែលខ្លួន Build ឡើង មានបញ្ហាត្រូវ Debug យើងម៉េច ត្រូវ Deploy យើងម៉េច ដើម្បីឲ្យអ្នកមកក្រោយមើលការខុស ត្រូវបន្ត។
